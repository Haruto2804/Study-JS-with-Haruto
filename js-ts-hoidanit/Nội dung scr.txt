1. Cài đặt công cụ
- Tải Visual Studio Code:
link: https://code.visualstudio.com/
- Setting format code, để Ctrl + S, để làm code trình bày đẹp, dễ nhìn hơn khi save File.
Cách làm: File/Preferences/Setting, trên thanh Search settings gõ format, click chọn Editor: Format On Save
- Cài extension: Auto Complete Tag: gợi ý cú pháp HTML/XML, Code Spell Checker: kiểm tra chính tả code
